## Project

I have built a small project for the talk, find the source code on [GitHub.](https://github.com/coderbootcamp/react-demo) The working demo is [here.](https://demowithreact.netlify.app/)

### Steps to set up the project on your system

- [Download](https://nodejs.org/en/) and install NodeJs in your system.
- Clone or download the project from GitHub.
- From the command prompt, go to the project directory and run ```npm start```
- You can also follow the instruction steps in the GitHub repository.

### Create New React Project

To create a new project, *create-react-app* is the best starting point. [Refer to this link.](https://reactjs.org/docs/create-a-new-react-app.html) I highly recommend reading the docs thoroughly.
